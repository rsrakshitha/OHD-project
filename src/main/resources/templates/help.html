Help - Online Help Desk System

1. Registration: First create an account using the Register page.

2. Login: Use your email and password to login.

3. Create Request: Go to Create Request and submit your issue.

4. View Requests: Check all your requests in the List page.

5. Track Status: Status will be OPEN, IN PROGRESS, or CLOSED.

6. Assignment: Admin will assign your request to staff.
